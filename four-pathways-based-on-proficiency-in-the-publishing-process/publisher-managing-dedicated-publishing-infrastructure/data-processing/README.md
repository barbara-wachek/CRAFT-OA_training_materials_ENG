# Data Processing

Data processing is a series of operations performed on raw data to obtain useful information. It can include filtering, transformation, aggregation, analysis and interpretation of data. In the work of publishing houses and repositories, data processing is an important step in managing information on scientific articles, citations, authors and reviews.
